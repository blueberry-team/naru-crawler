# 샘플링 규칙

## 기본 동작

`GET /api/dev/jobs/drafts` 로 전체 DRAFT 목록을 가져온 뒤 아래 기준으로 N건을 선택한다.

## 입력 방식

- `개수=N` — N건 샘플링
- `jobIds=1153,1154` — 지정한 ID만 리뷰
- `company=hitachi 개수=N` — 특정 회사의 DRAFT에서 N건

jobIds 가 지정되면 샘플링 규칙을 건너뛰고 해당 공고만 리뷰한다.

## 샘플링 우선순위

1. 아직 한 번도 리뷰하지 않은 공고 (createdAt 기준 오래된 것부터)
2. 회사·직군 다양성 — 같은 회사, 같은 position 이 연속되지 않도록 분산
3. 최근 적재된 공고

## 최소 기준

- 기본 샘플 수: 5건
- DRAFT 가 5건 미만이면 전부 리뷰한다.
- 사용자가 개수를 지정하면 그 값을 따른다.
