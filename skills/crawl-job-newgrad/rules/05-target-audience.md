# targetAudience 규칙

`targetAudience` 는 카드 상단에서 “이 공고가 누구 대상인지”를 보여주는 1줄 요약이다.

## 목적

- `joinDate` 와 역할이 다르다.
- `targetAudience` 에는 아래 정보만 들어간다.
  - 졸업 시기
  - 졸업 연도
  - 기졸 가능 여부

## 강한 규칙

- 명시된 정보만 사용한다.
- `対象:` 또는 `入社時期:` 같은 라벨을 붙이지 않는다.
- 같은 의미의 표현은 통일한다.
- `（既卒可）` 는 명시된 경우만 붙인다.
- 아무 정보도 없을 때만 `新卒`
- `卒業（予定/卒業済）` 같은 슬래시 표현은 사용하지 않는다.

## 추출 대상

- 졸업 시기
  - 예: `2027年3月までに卒業予定`
- 졸업 연도
  - 예: `27卒`, `2027卒`, `2027年度卒`
- 기졸 가능
  - 예: `既卒可`, `既卒者も可`, `第二新卒可`

아래 정보는 넣지 않는다.

- 경력 연수
- 동등 학위
- 비자 조건
- 어학 요건
- 자격증

이런 정보는 `targetCandidate` 로 보낸다.

## 출력 형식

1. 졸업 시기가 명시된 경우
   - 기졸 가능 없음:
     - `{YYYY年M月}までに卒業予定の方`
   - 기졸 가능 있음:
     - `{YYYY年M月}までに卒業予定の方（既卒可）`
2. 졸업 시기는 없고 졸업 연도만 있는 경우
   - 기졸 가능 없음:
     - `{YYYY卒}対象`
   - 기졸 가능 있음:
     - `{YYYY卒}対象（既卒可）`
3. 아무 정보도 없는 경우
   - `新卒`

## 표현 통일 규칙

- `27卒` / `2027卒` / `2027年度卒` / `2027年度卒業予定` → `2027卒`
- `卒業見込み` / `卒業予定` → `卒業予定`
- `既卒可` / `既卒者可` / `第二新卒可` / `既卒歓迎` → `既卒可`
- `2027年3月卒業見込み` / `2027年3月までに卒業予定` → `2027年3月までに卒業予定`
