# companySourceUrls 규칙

## 목적

- 어떤 공식 페이지를 근거로 회사 정보를 작성했는지 보여주는 용도다.
- 재수집을 위한 메타데이터는 아니다.

## 구조

- `Map<String, List<String>>`
- 값은 URL 배열
- 중복 URL 제거
- 최종 payload 에서는 한국어 키만 사용한다.

## 금지

- OpenWork URL 금지
- `openwork.jp` 를 포함한 URL 금지
- 허용되지 않은 키 이름 금지

## 허용 키

- `홈`
- `회사개요`
- `채용`
- `모집요강·복리후생`
- `기업이념`
- `IR`
- `기타`

## 보조 규칙

- 같은 URL을 여러 키에 중복해서 넣지 않는다.
- 한 URL은 실제로 근거로 사용한 섹션에만 넣는다.
