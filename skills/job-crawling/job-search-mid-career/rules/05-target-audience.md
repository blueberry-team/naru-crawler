# targetAudience 규칙

`targetAudience` 는 카드 상단에서 “이 공고가 누구 대상인지”를 보여주는 1줄 요약이다.

## 목적

- `joinDate` 와 역할이 다르다.
- `targetAudience` 에는 아래 정보만 들어간다.
  - 경력 연차
  - 직무
  - 레벨 / 시니어리티
  - 경력자 대상 여부

## 강한 규칙

- 명시된 정보만 사용한다.
- `対象:` 또는 `入社時期:` 같은 라벨을 붙이지 않는다.
- 같은 의미의 표현은 통일한다.
- 별도 대상 조건이 없더라도 이 스킬의 기본 대상은 중도채용이므로 최소값으로 `경력직` 을 사용한다.
- 기술 스택, 언어, 자격증, 비자, 근무지, 급여는 넣지 않는다.
- 단, `title` 과 완전히 같은 문장을 반복하지 않는다.
- `title` 에 없는 대상 조건이 보이면 `targetAudience` 에 우선 반영한다.

## 추출 대상

- 경력 연차
  - 예: `3年以上`, `5年程度`, `実務経験必須`
- 직무
  - 예: `バックエンド`, `プロダクトマネージャー`, `法人営業`
- 레벨 / 시니어리티
  - 예: `リーダー候補`, `マネージャー`, `シニア`
- 경력자 대상 표현
  - 예: `経験者`, `中途`, `キャリア採用`

아래 정보는 넣지 않는다.

- 세부 기술 스택
- 비자 조건
- 어학 요건
- 자격증
- 상세 필수 / 우대 조건

이런 정보는 `targetCandidate` 로 보낸다.

## 출력 형식

1. 직무 + 연차가 명시된 경우
   - `{직무} 경력 {N}년 이상`
2. 직무 + 레벨이 명시된 경우
   - `{직무} 경력자(리더급)`
   - `{직무} 경력자(매니저급)`
3. 직무만 명시된 경우
   - `{직무} 경력자`
4. 별도 조건 없이 중도채용만 명확한 경우
   - `경력직`

## 표현 통일 규칙

- `3年以上` / `3年以上の経験` / `実務経験3年以上` → `경력 3년 이상`
- `経験者` / `中途` / `キャリア採用` → `경력자`
- `リーダー候補` → `리더급`
- `マネージャー候補` / `管理職候補` → `매니저급`

## title 과의 분리 원칙

- `title` 은 공식 직무명 / 포지션명 / 레벨명을 유지한다.
- `targetAudience` 는 title 에 없는 “지원 대상 조건”을 요약한다.
- title 에 직무명과 레벨 정보만 있고, 별도 대상 조건이 없다면:
  - `targetAudience = "경력직"`
- title 과 별도로 연차 조건이 있으면:
  - 연차 조건을 `targetAudience` 에 반영한다.

예시:

- 제목: `バックエンドエンジニア（リーダー候補）`
  - `title = "バックエンドエンジニア（リーダー候補）"`
  - `targetAudience = "경력직"`

- 제목: `バックエンドエンジニア（リーダー候補）`
  - 본문: `実務経験3年以上`
  - `title = "バックエンドエンジニア（リーダー候補）"`
  - `targetAudience = "경력 3년 이상"`
