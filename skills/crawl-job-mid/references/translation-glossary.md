# 번역 glossary

사용자 노출 텍스트는 일본어에서 한국어로 번역한다. enum, 숫자, URL, slug, payload key는 번역하지 않는다.

## 전역 규칙

- glossary 치환은 일본어 원문 기준으로 1회 적용한다.
- forbidden 번역이 나오면 승인된 번역으로 교체한다.
- 한국어는 자연스럽게 쓴다. 번역투를 피한다.
- 고유명사, 공식 시험명, 제품명은 필요한 경우만 원문을 유지한다.
- `publishStatus`, `companySlug`, `position`, `locations`, `experienceLevel`, `workType`, `salaryMin`, `salaryMax`, `deadline`, `jobSourceUrl` 는 번역하지 않는다.
- `joinDate` 는 사용자 노출 필드이므로 한국어로 정규화한다. 자세한 포맷은 [`../rules/03-join-date.md`](../rules/03-join-date.md) 참조.
- ISO 날짜(`YYYY-MM-DD`) 는 번역하지 않는다.
